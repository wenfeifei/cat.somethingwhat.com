项目说明

----------------------------------------------------------
Cat.Web:
	Areas/Api					管理后台数据接口		对应站点 http://book.admin.somethingwhat.com
	Areas/Book					小程序前端数据接口		对应站点 http://book.somethingwhat.com
	Areas/Web					--						没什么用，只是一些说明页面

	Configs						配置文件目录
	log							日志文件

----------------------------------------------------------
Cat.Modules:
	Book						业务模块				包含喵喵看书小程序的业务代码逻辑和实体
	Cache						缓存模块				提供数据缓存功能，可通过配置文件来配置选择 MemoryCache 还是 Redis 缓存
	Log							日志模块				提供日志功能，可通过配置文件来配置选择 MongoDB 还是 log4Net
	Public						通用业务模块			提供一些公共业务模块功能，例如管理后台的账户管理、接口白名单管理、微信支付等

----------------------------------------------------------
Cat.Utility:
	Helper						帮助类
	Extensions					扩展方法

----------------------------------------------------------
Cat.Foundation:
	Config						配置文件实体			提供了对配置文件的统一访问
	CatContext.cs				上下文对象				提供了项目的环境配置信息、http上下文对象

----------------------------------------------------------
Cat.Enums:						枚举及系统常量

----------------------------------------------------------
